# AGENTS.md

## 프로젝트 설명
이 저장소는 CSV 파일을 기반으로 동작하는 Python CLI 가계부 앱 프로젝트다.

## 코딩 규칙
- 모든 함수와 메서드에는 타입 힌트를 필수로 사용한다.
- 함수 하나의 길이는 최대 50줄을 넘기지 않는다.

## TDD 규칙
- 구현 전에 반드시 테스트를 먼저 작성한다.

## 품질 규칙
- 순환복잡도는 10 이하로 유지한다.

## 품질 검수 규칙
- 커밋 전에 반드시 `qa_engineer` 서브에이전트로 품질 검수를 수행한다.

## 테스트 실행 명령어
- `pytest`
- `radon cc`

## 커밋 규칙
- 하나의 기능이 개발되면 즉시 커밋하고 푸시한다.
